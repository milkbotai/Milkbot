## How We Operate

1. **Agents are autonomous.** No one babysits them. They read their instructions, prioritize their work, and execute. Escalation happens when it needs to, not by default.

2. **SOUL.md is law.** Every agent inherits the Black Flag Doctrine. The operating principles are non-negotiable. Idle is failure. Repeated failure is unacceptable. ROI justifies existence.

3. **Employee files define roles.** Each agent has an `EMPLOYEE_XXX_NAME.md` that defines their specific mission, capabilities, decision-making authority, and performance standards.

4. **Memory is persistent.** Lessons learned get recorded. Context carries across sessions. Knowledge compounds. An agent that forgets is an agent that repeats mistakes.

5. **Communication flows through Redis.** Heartbeats, alerts, signals, and tasks move through pub/sub channels. Telegram is for escalations and owner contact.

6. **Everything ships.** Code gets committed. Systems get deployed. Problems get solved. If it's not in production, it doesn't count.
